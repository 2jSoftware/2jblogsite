Accessibility Statement

The Global Development Research Portal is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone, and applying the relevant accessibility standards.

Conformance Status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

The Global Development Research Portal is striving to conform to WCAG 2.1 level AA. We are working to achieve this conformance, and this statement will be updated as we improve.

Compatibility with Browsers and Assistive Technology

The Global Development Research Portal is designed to be compatible with the following assistive technologies:

  • Screen readers (including NVDA, JAWS, VoiceOver, and Narrator)
  • Screen magnifiers
  • Speech recognition software
  • Keyboard-only navigation

The website is designed to be compatible with the following browsers:

  • Latest versions of Chrome, Firefox, Safari, and Edge
  • Internet Explorer 11 and above

Accessibility Features

The Global Development Research Portal includes the following accessibility features:

  • Keyboard accessibility: All interactive elements are accessible via keyboard
  • Skip links: Allow users to bypass navigation and jump to main content
  • Semantic structure: Proper use of headings, landmarks, and semantic HTML
  • Text alternatives: All non-text content has text alternatives
  • Color contrast: Text meets minimum contrast requirements
  • Resizable text: Text can be resized up to 200% without loss of functionality
  • Focus indication: Visible focus indicators for keyboard navigation

Known Limitations

While we strive to ensure the entire website is accessible, we are still working on some aspects:

  • Some older PDF documents may not be fully accessible. We are working to remediate these files.
  • Some third-party content may not be fully accessible. We are working with our partners to improve this.
  • Complex data visualizations are being improved to ensure they have appropriate text alternatives.

Technical Specifications

Accessibility of the Global Development Research Portal relies on the following technologies:

  • HTML5
  • WAI-ARIA
  • CSS3
  • JavaScript

Assessment Approach

The Global Development Research Portal has been assessed in the following ways:

  • Self-evaluation using the Web Content Accessibility Guidelines (WCAG) 2.1
  • Testing with screen readers
  • Keyboard-only navigation testing
  • Automated testing using accessibility evaluation tools
  • User testing with people with disabilities (planned)

Improvement Plan

We are committed to continuous improvement of our website's accessibility. Our accessibility improvement plan includes:

  • Regular accessibility audits and testing
  • Ongoing staff training on creating accessible content
  • Implementation of user feedback
  • Periodic review and update of this accessibility statement
